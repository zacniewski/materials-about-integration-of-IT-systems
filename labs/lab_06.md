# Laboratorium 6: Projekt końcowy – zintegrowany system z CI/CD

## Czas trwania: 10 godzin

### Cel:
Samodzielne zaprojektowanie i wdrożenie kompletnego systemu informatycznego zintegrowanego przy użyciu poznanych technologii (Git, Docker, PaaS, GitHub Actions).

### Zadania i ćwiczenia:
1. **Zdefiniowanie architektury (1h):**
   - Wybór technologii (Frontend, Backend, DB).
   - Szkic schematu integracji.

2. **Implementacja i konteneryzacja (3h):**
   - Przygotowanie kodu aplikacji.
   - Stworzenie plików `Dockerfile` i `docker-compose.yml` dla środowiska deweloperskiego.

3. **Konfiguracja CI/CD (4h):**
   - Przygotowanie workflow w GitHub Actions.
   - Automatyzacja budowania obrazu Docker.
   - Automatyzacja wdrożenia na platformę PaaS.

4. **Testowanie i dokumentacja (2h):**
   - Testy integracyjne całego systemu.
   - Przygotowanie dokumentacji w pliku `README.md` (instrukcja uruchomienia, opis usług).

### Wymagania na zaliczenie:
- Repozytorium na GitHub z pełną historią zmian.
- Działająca aplikacja w chmurze (PaaS).
- Poprawnie skonfigurowany i zielony potok CI/CD.
- Prezentacja projektu prowadzącemu.
